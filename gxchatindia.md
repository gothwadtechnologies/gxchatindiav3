# GxChat India

GxChat India is a high-performance, real-time messaging and social media application designed for the Indian market. It combines the best features of modern communication apps with a clean, intuitive interface and robust security.

## ✨ Key Features

- **Real-time Chat**: Instant message delivery, read receipts, and live typing indicators.
- **Social Networking**: Share posts, follow friends, and explore short-form video content (Reels).
- **App Security**: Global App Lock with PIN or Password protection and advanced privacy settings.
- **User Profiles**: Custom profiles with bios, status updates, and real-time presence indicators.
- **Responsive Layout**: Optimized mobile-first design with a centered, modern interface.
- **Push Notifications**: Stay updated with real-time alerts for messages and interactions.

## 🛠 Technology Stack

- **Frontend**: React 19, Vite, TypeScript.
- **Styling**: Tailwind CSS 4 for a modern, utility-first UI.
- **Backend**: Firebase (Firestore, Auth, RTDB) for real-time data and presence.
- **Server**: Express for API extensions and notification handling.
- **Animations**: Motion (Framer Motion) for smooth, fluid transitions.
- **Icons**: Lucide React for consistent iconography.

## 🇮🇳 Vision

GxChat India aims to provide a secure, fast, and reliable communication platform for users across India, focusing on privacy, performance, and a superior user experience.
